# Government Schemes RAG Chatbot

A bilingual (English/Hindi) chatbot that helps Indian citizens find information about government schemes using RAG (Retrieval Augmented Generation).

## Tech Stack
- **LLM:** OpenAI GPT-4o-mini
- **Embeddings:** OpenAI text-embedding-3-small
- **Vector Store:** FAISS
- **Framework:** LangChain
- **UI:** Gradio

## Schemes Covered (20)
PM Kisan, Ayushman Bharat, PM Awas Yojana, Ujjwala, Sukanya Samriddhi, Atal Pension, MUDRA, Ladli Yojana (Delhi), Free Bus for Women (Delhi), and more!

## Features
- Ask questions in English or Hindi
- Covers 20+ Central and Delhi government schemes
- Instant AI-powered answers
